# 👨‍💻 Personal Profile Page (Semantic HTML)

A simple and well-structured **Personal Profile Page** built using **Semantic HTML**. This project demonstrates the use of meaningful HTML elements to create an organized and accessible webpage.

---

## 🚀 Features

✅ Semantic HTML Structure

✅ Responsive Meta Tags

✅ Personal Introduction Section

✅ Skills Section with Unordered List

✅ Contact Information Section

✅ LinkedIn Profile Link

✅ Clean and Beginner-Friendly Code

## 📋 Sections Included

### 🏷️ Header

* Name
* Professional Title

### 🙋 About Me

* Personal introduction
* Educational background
* Frontend and Backend development interests

### 🛠️ Skills

* HTML
* CSS
* JavaScript
* Java
* SQL

### 📞 Contact

* Email Address
* Phone Number
* LinkedIn Profile

## 💻 Technologies Used
